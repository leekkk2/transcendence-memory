# Guide Index

为 LLM Agents 和 operator 提供可直接抓取的部署/交付入口。

## For LLM Agents
- `docs/guide/installation.md`
- `docs/guide/backend-deployment.md`
- `docs/guide/frontend-handoff.md`
- `docs/guide/auth-handoff.md`

## Intent
这些文档应支持类似下面的使用方式：

```bash
curl -s <raw-url-to-guide>
```

也就是说，agent 不需要先通读整个仓库 README，直接抓对应 guide 就能执行当前阶段需要的动作。
